# WorkSure

WorkSure is a comprehensive parametric insurance and gig-economy management platform. It consists of a Web Admin dashboard and a Mobile application for gig workers.

## Project Structure

- `mobile/`: Mobile application built with React and Hono (backend).
  - `frontend/`: React frontend for the mobile app.
  - `backend/`: Hono backend for the mobile app.
- `web/`: Web Admin application.
  - `frontend/`: React frontend for the admin panel.

## Features

- **Parametric Insurance**: Automated claims based on real-time weather and environmental data.
- **Risk Pulse**: Live monitoring of environmental risks for gig workers.
- **Wallet & Claims**: Seamless financial management and automated claim processing.
- **Profile Builder**: Personalized onboarding for different gig personas.

## Setup

1. **Mobile Frontend**: `cd mobile/frontend && npm install && npm run dev`
2. **Mobile Backend**: `cd mobile/backend && npm install && npm run dev`
3. **Web Frontend**: `cd web/frontend && npm install && npm run dev`
